# Operator Guide

## First Deployment
1. Register operator, production lines, process specifications, staff
   and robots.
2. Import existing process-safety and production history.
3. Run read-only specification-scope and production/inspection robot
   mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run release record and audit export.

## Minimum Production Controls
- process-specification-scope validation before any production run
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (batch release
  without verification, safety-critical process changes)
- evidence-backed release records
- audit export for every dispatch, sign-off and release
- backup manual process-safety review

## Certification
Certified operators must prove robot-safety integrity, process-safety
discipline, evidence-backed release records and human review for
release-affecting actions.
